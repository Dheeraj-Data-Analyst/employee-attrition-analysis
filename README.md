# 📊 Employee Attrition Analysis (Excel Project)

## 📌 Project Overview

This project analyzes employee attrition using Microsoft Excel. The goal is to identify patterns and key factors contributing to employee turnover.

## 🎯 Objective

* Analyze employee attrition trends
* Identify factors affecting employee retention
* Provide insights using Excel dashboards

## 📂 Dataset

* File Name: `Employee Attrition.xlsx`
* Data includes employee details like age, department, salary, job role, and attrition status

## 🛠️ Tools Used

* Microsoft Excel

  * Pivot Tables
  * Charts (Bar, Pie, Line)
  * Conditional Formatting
  * Data Cleaning

## 📊 Analysis Performed

* Attrition by Department
* Attrition by Age Group
* Salary vs Attrition
* Job Role-wise Attrition

## 📈 Key Insights

* Higher attrition observed in lower salary range
* Certain departments show higher turnover
* Younger employees tend to leave more frequently
